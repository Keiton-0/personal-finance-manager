# Personal Finance Manager

A desktop application developed in Python for managing personal monthly finances.

## Features

- Add income
- Add expenses
- Real-time balance updates
- Data saved automatically using JSON
- Modern GUI with Tkinter
- Persistent storage

## Technologies Used

- Python
- Tkinter
- JSON
- Git & GitHub
